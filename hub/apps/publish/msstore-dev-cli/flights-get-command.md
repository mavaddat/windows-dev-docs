# Flights - Get Command

Retrieves a flight for the specified Application and flight.

## Usage

```console
msstore flights get <productId> <flightId>
```

## Arguments

| Argument    | Description |
|-------------|-------------|
| `productId` | The product ID. |
| `flightId` | The flight ID. |

## Help

```console
msstore flights get --help
```

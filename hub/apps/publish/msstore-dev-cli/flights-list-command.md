# Flights - List Command

Retrieves all the Flights for the specified Application.

## Usage

```console
msstore flights list <productId>
```

## Arguments

| Argument    | Description |
|-------------|-------------|
| `productId` | The product ID. |

## Help

```console
msstore flights list --help
```
